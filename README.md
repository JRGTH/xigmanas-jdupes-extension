**Description:**

 This is the XigmaNAS Jdupes Extension for embedded platforms, Powerful duplicate file finder and an enhanced fork of ‘fdupes’.

jdupes is a program for identifying and taking actions upon duplicate files. It is a fork of fdupes, but not a drop-in replacement.

See http//www.virkki.com/jyri/articles/index.php/dupd-vs-jdupes-take-2/ for a comparision to dupd.

WWW https//github.com/jbruchon/jdupes
